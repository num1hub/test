# Capsule Graph Snapshot

Use this skill to get a compact picture of the current capsule graph before choosing a target area.

## When To Use It

- Before graph cleanup or refactor work.
- Before introducing new foundation capsules or project hubs.
- When you need to identify the most connected or structurally central capsules.

## How To Use It In N1Hub

- Prefer the `capsule_graph_snapshot` internal tool in Symphony or N-Infinity.
- If you are outside that runtime, inspect the relevant capsules directly and keep the scope tight.

## Rules

- Use the snapshot to orient yourself, not as a substitute for reading the actual target capsules.
- After identifying a target area, dig down into the concrete files that define it.
